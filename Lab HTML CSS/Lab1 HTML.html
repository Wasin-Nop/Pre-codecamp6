Similarity & Differentiation

Tags Similarity Differentiation
hr & br ใช้คั่นข้อมูล ทำให้เกิดการแบ่งระหว่างบรรทัด hr ใช้ขีดเส้นคั่นระหว่างหน้า br เกินช่องว่างระหว่างบรรทัด
ul & ol ใช้ลำดับข้อมูล ul ลิสไม่มีอันดับ ol เป็นลิสมีลำกับ 1,2,3....n
select & checkbox & radio button เป็นช่องรับข้อมูล select เป็นช่องรับข้อมูลแบบ drop down list
checkbox ช่องรับข้อมูล เลือกได้มากกว่า 1 ช่อง
radio button ช่องรับข้อมูล เลือกได้แค่ช่องเดียว
div & span ใช้สำหรับกำหนดรูปแบบหรือลักษณะข้อความที่อยู่ภาย tag มักใช้ร่วมกับ CSS div เป็น tag ที่มีไว้แบ่งส่วนเนื้อหาของ HTML ออกจากกันเป็นส่วนๆ (block) ถ้าไม่ได้กำหนดคุณลักษณะใดๆ ก็จะมีผลเหมือนกับการใช้ tag p และเมื่อแสดงผลแล้วจะขึ้นบรรทัดใหม่ให้อัตโนมัติ
span แต่มักใช้จัดแต่ง ข้อความสั้นๆ (inline) ที่อยู่ใต้การกำหนดรูปแบบของ tag อื่นอยู่แล้ว เพื่อให้มีรูปแบบที่ต่างไปจากรูปแบบเหล่านั้น และจะไม่ขึ้นบรรทัดใหม่
script & style เป็น tag ภายใต้ html script ใช้สำหรับการเขียน Javascript
style ใช้สำหรับการเขียน CSS

Input Type

Tags Input Type
button กำหนดปุ่มคลิกได้ (ส่วนใหญ่ใช้กับจาวาสคริปต์เพื่อเปิดใช้งานสคริปต์)
checkbox กำหนดช่องทำเครื่องหมาย เลือกได้มากกว่า 1
color กำหนดตัวเลือกสี
date กำหนดช่องกำหนดปี เดือน และวัน (ไม่มีเวลา)
datetime-local กำหนดวันที่และเวลาการควบคุม (ปีเดือนวันชั่วโมงนาทีวินาทีและเสี้ยววินาที (โซนเวลา)
email กำหนดเขตข้อมูลที่อยู่อีเมล
file กำหนดข้อมูลไฟล์และเลือกปุ่ม "Browse ... " (สำหรับการอัปโหลดไฟล์)
hidden กำหนดช่องใส่ที่ซ่อนอยู่
image กำหนดภาพเป็นปุ่มส่ง
month กำหนดรับข้อมูลเดือนและปี
number กำหนดช่องสำหรับป้อนตัวเลข
password กำหนดฟิลด์รหัสผ่าน
radio กำหนดปุ่มรับข้อมูล เลือกได้แค่ช่องเดียว
range กำหนดควบคุมสำหรับการป้อนจำนวนที่มีค่าที่แน่นอนไม่ได้เป็นสิ่งที่สำคัญ (เช่นการควบคุมเลื่อน)
reset กำหนดปุ่มรีเซ็ต (รีเซ็ตค่าทุกรูปแบบไปเป็นค่าเริ่มต้น)
search กำหนดฟิลด์ข้อความสำหรับการป้อนสตริงการค้นหา
submit กำหนดปุ่มส่ง
tel กำหนดฟิลด์สำหรับป้อนหมายเลขโทรศัพท์
text ค่าเริ่มต้น. กำหนดฟิลด์ข้อความบรรทัดเดียว (กว้างเริ่มต้นคือ 20 ตัวอักษร)
time กำหนดควบคุมสำหรับการป้อนเวลา (ไม่มีโซนเวลา)
url กำหนดฟิลด์สำหรับป้อน URL ที่
week กำหนดสัปดาห์และปี Control (ไม่มีโซนเวลา)

label ใช้กำหนดป้ายชื่อให้กับ ช่องรับข้อมูล
del ขีดเส้นทับตัวอักษร
b ตัวอักษรหนาขึ้น
em เน้นตัวอักษร(ทำให้เอียง)
mark ไฮไลท์ตัวอักษร
i เอียงตัวอักษร
sub ตัวอักษรชิดบรรทัดด้านล่าง
sup ตัวอักษรชิดบรรทัดด้านบน
